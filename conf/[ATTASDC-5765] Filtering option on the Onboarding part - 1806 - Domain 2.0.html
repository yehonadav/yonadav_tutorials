        Description

        As a Onboarding user I want to ability to filter specific VLMs/VSPs according to specific filtering criteria.

        The Onboarding filtering should be part of both Workspace and Catalog (where in Catalog it is part ALL and BY BENDOR tab options)

        In order to achieved it we need to add a new left navigation panel on both Onboarding Workspace and Onboarding Catalog.

        The filtering criteria are:

        Role:

        • Owner
        • Contributor

        Entity Type:

        • VLM
        • VSP

        Category: (At the moment not part of the filtering until we will get the approval)

        • Optional categories

        Onboarding Procedure:

        • Network package
        • Manually

        Each option will have a radio buttons that will allow the user select its prefer filter option.

        Each click on a filter will filter out all the VSPs/VLMs that do not belongs to the filter,

        Each click that remove a filter will change the display and display back the filtered out VSPs/VLMs

        User should be able to select multiple filters (even on the same filtering criteria category, e.g. select both VLM and VSP)
